# Hydra PeptideOS

AI-powered peptide protocol operating system with Samurai Men-Yoroi inspired design.

## Features
- Biomarker Tracking
- AI-Powered Peptide Protocols  
- 12-Week Cyclic Protocol Management
- Longitudinal Trend Analysis
- Safety Guardrails
- Clinical PDF Reports

## Tech Stack
- React 18 + Vite + TypeScript
- Tailwind CSS with Samurai theme
- Express.js backend
- Shadcn UI components

## Theme
Crimson Red (#DC143C) and Fiery Orange (#FF4500) inspired by traditional Japanese samurai armor.
